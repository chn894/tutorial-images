# tutorial-images

Image repository for tutorial and documentation assets used by `chn894`.

## Suggested structure

- `images/tutorials/`: tutorial screenshots and step-by-step assets
- `images/misc/`: reusable supporting images

## Naming tips

- Use lowercase file names
- Prefer hyphens instead of spaces
- Group by topic or product

Example:

`images/tutorials/github/create-repo-step-01.png`
